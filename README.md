**Zenith** is a privacy-first startup page that evolves with your environment.

## Features

- **Atmospheric Backgrounds**: A generative engine that syncs with real-time weather, seasons, and time of day to create 100+ unique themes.
- **Deep Focus**: Includes a "Zen Mode" that fades out distractions and a built-in Pomodoro timer.
- **Privacy First**: Uses OpenMeteo (no API key required) and local storage. No data leaves your browser.
- **Productivity**: Daily focus task tracking and a Glassmorphic Speed Dial for quick access.

## Tech Stack

- **Framework**: React + TypeScript + Vite
- **Styling**: TailwindCSS (Zero-runtime)
- **Icons**: Lucide React
- **Animation**: CSS Keyframes + Transitions

## Getting Started

1.  Clone the repository.
2.  Run `npm install`.
3.  Run `npm run dev`.
